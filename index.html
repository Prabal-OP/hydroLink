IoT Water Level Monitoring System

Project Overview

The IoT Water Level Monitoring System is a smart solution developed at KLS Gogte Institute of Technology (GIT) that enables real-time water level monitoring through IoT technology. The system combines hardware sensors with a custom-built Android application to provide efficient water level tracking and management capabilities.

Technical Specifications

Hardware Components
Ultrasonic Sensor
  • Model: GPRobot A02YYU/W Waterproof
  • Range: 3-450cm
  • Interface: UART
  • Protection Rating: IP67
  • Purpose: Precise water level measurement
Microcontroller
  • Model: ESP32-WROOM-32E, ESP32-DevKitC-32E
  • Features:
  • Built-in WiFi and Bluetooth
  • Arduino compatible
  • Development board module
Power Supply
  • Model: Hi-Link HLK-PM01
  • Output: 5V/3W
  • Type: Switch Power Supply Module
  • Purpose: Stable power delivery to the system

Mobile Application

  • Development Platform: Android Studio
  • Real-time data fetching through API
  • User-friendly interface
  • Data visualization
  • Push notifications

Backend Integration

  • API-based data communication
  • Real-time data synchronization
  • Secure data transfer protocols

System Architecture

The system architecture consists of three integrated layers: a DFRobot ultrasonic sensor collects water level data, an ESP32 microcontroller processes and transmits this data via WiFi, and an Android application receives and displays the information through API integration.

Features: Real-time water level monitoring with push alerts, data tracking, and Android app-based remote access.

Benefits: Efficient water management with accurate monitoring, low maintenance, and weather-resistant operation.
